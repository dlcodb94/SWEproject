# SWEproject

- Conventional Commit 

    | **type** | **설명** |
    | --- | --- |
    | **feat** | **기능 개발과 관련** |
    | **docs** | **주석/ReadMe 등 문서화 관련** |
    | **test** | **테스트 관련** |
    | **fix** | **버그나 typo 등을 수정 사항 관련** |
    | **chore** | **코드와 관련이 없는 내용들을 수정할 때 (License 등)** |
    | **ci** | **CI/CD 등과 관련한 작업을 수행할 때** |
